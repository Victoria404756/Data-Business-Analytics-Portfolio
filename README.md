# Business Analytics Portfolio

## About

This portfolio showcases business analytics projects focused on retail sales performance, profitability analysis, customer segmentation, regional performance, dashboard development, and executive reporting.

The projects demonstrate practical skills in Tableau, PostgreSQL, SQL, Excel, Power BI, DAX, data modeling, KPI reporting, and business insight communication.

> These projects demonstrate practical business analytics skills relevant to Data Analyst, Business Analyst, Reporting Analyst, and BI Analyst roles.

Author: Victoria Zhang

## Project 1: Retail Sales Performance Dashboard – Tableau

This project presents an interactive Tableau dashboard using the Sample Superstore dataset. The dashboard analyses sales, profit, customer segments, product categories, and regional performance.

Key deliverables include:

- Tableau dashboard
- Business Requirements Document
- Executive Summary
- Business insights and recommendations

Skills demonstrated:

- Tableau dashboard development
- KPI design
- Data visualization
- Sales and profitability analysis
- Executive reporting

## Project 2: Retail Sales Analysis – SQL & Excel

This project uses PostgreSQL and Excel to analyse retail sales performance from the Sample Superstore dataset. SQL queries were used to calculate KPIs, analyse product performance, evaluate customer segments, review regional performance, and identify business recommendations.

Key deliverables include:

- PostgreSQL SQL scripts
- SQL views for Excel export
- Excel reporting dashboard
- Executive Summary

Skills demonstrated:

- SQL querying
- PostgreSQL analysis
- Excel reporting
- KPI calculation
- Customer segmentation
- Product profitability analysis
- Regional performance analysis
- Business recommendation writing
