## About

This portfolio showcases business analytics and data analytics projects focused on retail sales performance, profitability analysis, customer segmentation, regional performance, dashboard development, predictive modeling, and executive reporting.

The projects demonstrate practical skills in Tableau, PostgreSQL, SQL, Excel, Power BI, DAX, Power Query, Python, machine learning, customer segmentation, KPI reporting, and business insight communication.

> These projects demonstrate practical business analytics skills relevant to Data Analyst, Business Analyst, Reporting Analyst, BI Analyst, and Marketing Analytics roles.

## Project 4: Census Income Classification & Marketing Segmentation

This project applies machine learning and segmentation analysis to census data to identify income patterns and support marketing targeting strategies. The project combines predictive modeling with interpretable population segmentation to help prioritize high-value audiences and tailor marketing approaches.

The project includes an income classification model and segmentation analysis based on demographic, employment, household, and financial attributes.

### Key Deliverables

- Income classification model
- Population segmentation analysis
- Model evaluation
- Marketing targeting insights
- Business-focused final report

### Skills Demonstrated

- Python data analysis
- Machine learning classification
- Customer and population segmentation
- Marketing analytics
- Feature analysis
- Model evaluation
- Business insight communication
- Predictive analytics
